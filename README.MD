#  Rick & Morty Character Explorer

Aplicação web que consome a API de **Rick and Morty** para buscar e exibir personagens de forma dinâmica e interativa.

---

##  Funcionalidades

- Busca de personagens pelo nome  
- Exibição dinâmica em cards  
- Modal com detalhes completos do personagem  
- Paginação entre resultados  
- Tratamento de erros (personagem não encontrado / falha de conexão)

---

##  Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Fetch API  

---

##  Como funciona

A aplicação realiza requisições à API pública de Rick and Morty, trazendo dados em tempo real.  
Os personagens são exibidos dinamicamente na tela utilizando manipulação de DOM com JavaScript.

---

Link para a API: https://rickandmortyapi.com/api/character
